# Goal of this project

Side project to learn and vibe an aesthetic looking responsive website with elegant non pragmatic choices.

Goal : create a today I learned application. I learn something, I want to dump it somewhere and find them later !

Aesthetic : riced terminal feeling. Straight outta hyprland boldest designs.

Stack : Vanilla React, just for fun.

Features to dig into : dynamic design system, design system building, connection to llms

# Non pragmatic features to open the system to

- Auto tagging learnings with LLMs
- Fancy learning graphs
- TIL post and sharing
- Giga plugin ecosystem on storage : start simple with local storage, then create connectors for indexedDb, gdrive, firebase, whatever implements the interface.
- LLM recap of learnings of the week (same must be super easy to plug to any LLM provider)
